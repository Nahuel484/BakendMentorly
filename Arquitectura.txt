handlers/routes.go          → Lógica de login
services/auth_service.go    → Funciones de autenticación
services/errors.go          → Errores personalizados
```

Cuando necesites agregar más pantallas como:
- Búsqueda de mentores
- Gestión de solicitudes
- Sistema de mensajería
- Perfil del usuario avanzado

Simplemente creas nuevos archivos sin contaminar el código existente:
```
handlers/
├── auth_handler.go         (Login, registro)
├── mentor_handler.go       (Búsqueda de mentores)
├── request_handler.go      (Gestión de solicitudes)
└── message_handler.go      (Mensajería)

services/
├── auth_service.go
├── mentor_service.go
├── request_service.go
└── message_service.go
